# PET API

## Introduction

This is a simple (Tamagotchi like) API. It is meant to be used with my NodeJS Discord bot : [BOT-PET](https://github.com/msczk/bot-pet)

## Installation

Clone this repository

```cd api-pet```

```cp .env.example .env```

```php artisan migrate```

```php artisan db:seed```

```php artisan serve```
